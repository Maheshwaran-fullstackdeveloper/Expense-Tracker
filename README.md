# Expense Tracker - Full Stack Web Application

A premium, modern, and visually stunning Expense Tracker built with React, Node.js, and MongoDB.

## Features

- **Modern Dashboard**: High-quality data visualization with Recharts and glassmorphism design.
- **JWT Authentication**: Secure login/registration with access and refresh token management.
- **Transaction Management**: Effortlessly add, filter, and track daily income and expenses.
- **Savings Goals**: Plan for big purchases by setting targets, deadlines, and uploading product images.
- **Premium UI/UX**: Ultra-smooth animations powered by Framer Motion and styled with Tailwind CSS.
- **Mobile Responsive**: Fully optimized for desktop and mobile devices.

## Tech Stack

- **Frontend**: React (Vite), Tailwind CSS, Framer Motion, Recharts, Lucide React, Axios.
- **Backend**: Node.js, Express, MongoDB (Mongoose), JWT, Bcrypt, Multer.

## Getting Started

### Prerequisites

- Node.js installed.
- MongoDB instance (local or Cloud Atlas).

### Installation

1. **Clone the repository or download the files.**

2. **Setup Backend:**

   ```bash
   cd backend
   npm install
   ```

   - Configure `.env` with your `MONGODB_URI` and JWT secrets.

3. **Setup Frontend:**
   ```bash
   cd ../frontend
   npm install
   ```

### Running the Application

1. **Start Backend (Development):**

   ```bash
   cd backend
   npm run dev
   ```

2. **Start Frontend:**

   ```bash
   cd ../frontend
   npm run dev
   ```

3. **Access the app:** Open `http://localhost:5173` in your browser.
